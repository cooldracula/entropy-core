# PlantUML

### Installation

Install PlantUML extension for VS Code (requires Java JRE)

https://plantuml.com/

### how to use

fill any file with: 

```
@startuml <optional: name_outputfile>
<your UML code>
' this is a comment 
/' 
block comment 
'/
@enduml
```

* right-click -> "preview current diagram
* right-click -> "export"
* printed files will be printed to name_outputfile.png
